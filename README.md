## 快速开始

打开 `card.html` → 选择角色 → 点击「开始攀登」！

**操作方式：**
| 操作 | 方式 |
|------|------|
| 打牌 | 左键点击手牌 |
| 结束回合 | 点击「⏭ 结束」按钮 |
| 使用药水 | 按 `Q` 键 |
| 打开图鉴 | 按 `B` 键 |
| 打开地图 | 按 `M` 键 |
| 查看卡牌信息 | 战斗中按B查看全卡牌/状态/怪物图鉴 |
| 查看路线图 | 按M查看全楼层路线 |
| 跳过奖励 | 右键点击 |
| 离开商店 | 点击空白区域或右键 |

**快捷键：**
- `B` — 图鉴开关
- `M` — 地图开关
- `Q` — 使用药水

### 💡 新手提示
- **先选铁甲战士**上手，熟悉后再玩静默猎手和观者
- **前期**多拿优质普卡（耸肩、剑柄打击、铁斩波）
- **篝火**优先升级核心输出牌
- **卡组越精简越强**，商店移除低质量卡
- **保留药水给BOSS战**
- 战斗中按 **M** 看地图规划路线

> **上手难度：★☆☆☆☆**

**特点：** 简单粗暴，力量成长型。通过提升力量来增加每张攻击牌的伤害。多段攻击（双斩、剑刃回旋）从力量中获益极大。格挡体系完善（耸肩、哨卫、铁壁等），生存能力强。

- 消耗战：腐化+无惧疼痛+黑暗之拥（技能变0费并消耗，每消耗得甲+抽牌）

## 核心机制

### 回合流程
```
回合开始 → 重置能量至3 → 重置格挡（除非有壁垒）
        → 手牌debuff-1 → 触发能力效果（金属化/恶魔形态等）
        → 抽5张牌
        → **玩家回合**（打牌）
结束回合 → 弃手牌 → 触发山铜等
        → **敌方回合**（执行意图）
        → 中毒结算 → 获得新意图
        → 开始新回合
```

### 能量系统
- 每回合 3 点能量
- 卡牌左上角数字为费用，⚡表示能量
- **0费卡牌**可以免费打出（冲突、战斗专注、鲁莽等）
- 快乐花遗物每3回合+1能量上限

### 格挡系统
- 格挡（🛡️）值在回合开始时清零
- **壁垒**能力牌可让格挡保留
- 格挡完全吸收伤害后消失（不会穿透）
- 堑壕可将当前格挡翻倍
- 摔绊造成等于当前格挡的伤害

### 抽牌与牌库
- 每回合抽5张
- 抽牌堆为空时，弃牌堆洗入抽牌堆
- **消耗**的牌本场战斗不再出现
- **能力牌**打出后生效但不进入任何牌堆
- 伤口、晕眩等状态牌会污染牌库
- 铁靴（回收消耗牌）可从消耗堆回收

### 卡牌升级
- 在篝火处选择「升级」→ 选择一张牌
- 升级后的卡牌名称带 **+** 标记
- 数值提升约50%（攻击/格挡增加，效果增强）

| 图标 | 类型 | 说明 |
|:----:|------|------|
| ⚔️ | 普通战斗 | 标准战斗，掉落金币+卡牌奖励 |
| 💀 | 精英战斗 | 更强敌人，掉落遗物+更多金币 |

### 新手攻略

**第1幕（1-9层）：构建卡组**
- 优先拿高质量普卡（耸肩、剑柄打击、铁斩波）
- 篝火优先升级关键输出牌
- 精英怪掉落遗物，有实力就打
- 第5层商店前攒够金币
- BOSS前保证有充足格挡

**第2幕（10-18层）：成型发力**
- 卡组应有明确流派方向
- 精英怪变强，注意评估实力
- 商店移除低质量卡（打击、防御）
- 收集配合流派的关键金卡
- BOSS(守卫者)注意防御形态后的输出窗口

**第3幕（19-27层）：全力冲刺**
- 卡组应完全成型
- 精英怪很强，慎重挑战
- 保留药水应对BOSS
- 觉醒者/时间吞噬者需要持续高输出
- 最后一个篝火用于备战

### 金币管理
- 普通战斗：20-40金币
- 精英战斗：30-60金币
- 宝箱：额外25-40金币
- 商店卡牌：30-80金币
- 移除卡牌：50金币
- 生命药水：15-30金币

## Build 流派推荐

### 🔥 铁甲战士 Build

#### 1. 力量战 ⭐⭐⭐⭐⭐
**核心卡：** 燃烧 + 恶魔形态 + 双斩/剑刃回旋 + 识破  
**关键遗物：** 金刚杵 + 手里剑  
**打法：** 前期叠力量，后期多段攻击一发入魂  
**强度：** 成型后一回合轻松100+伤害

**推荐卡组（12-15张）：**
```
打击×2~3 → 燃烧 → 识破 → 双斩/剑刃回旋 ×2
恶魔形态 → 耸肩×2 → 铁斩波 → 剑柄打击
+ (任意过牌/防御)
```

#### 2. 格挡战 ⭐⭐⭐⭐
**核心卡：** 壁垒 + 堑壕 + 摔绊 + 岿然不动  
**关键遗物：** 山铜  
**打法：** 叠高格挡→堑壕翻倍→摔绊打出格挡值的伤害  
**强度：** 最安全的流派，百甲以上摔绊一发几百伤害

**推荐卡组：**
```
壁垒 → 堑壕 → 摔绊 ×2 → 耸肩×2 → 哨卫 → 铁斩波
岿然不动 → 剑柄打击 → 通用防御牌
```

#### 3. 腐化消耗流 ⭐⭐⭐⭐⭐

**推荐卡组：**

腐化 → 无惧疼痛 → 黑暗之拥

**打法：** 自伤触发撕裂成长，叠加恶魔形态

## Tips & 技巧

1. **卡组越精简越强** — 移除无用的打击、防御，让核心卡更容易抽到
2. **攻击 vs 防御的平衡** — 一般保持2:1的比例
3. **过早拿高费卡**会导致前期卡手
4. **精英怪**虽然难打但掉遗物，是滚雪球的关键
5. **保留药水给BOSS**，前期尽量不用
6. **图鉴系统**按B打开，可随时查看所有卡牌和敌人信息
7. **存档自动保存**，重新打开可「继续游戏」
8. **已升级的卡牌不可再次升级**，选好目标

## ❓ 常见问题

**Q: 没有战利品？**
A: 检查是否在打出最后一张牌时报错（死灵书、木乃伊手等遗物可能冲突）。刷新重试。

**Q: 手牌全是状态牌怎么办？**
A: 进化+火焰吐息可以化劣势为优势。商店可移除卡牌。

**Q: 观者的怒火姿态太脆了？**
A: 怒火是双刃剑，注意计算斩杀线，确保当回合能秒杀或快速退出姿态。

**Q: 怎么获得更多药水？**
A: 白兽雕像遗物保证每场掉落。商店也可购买。

**Q: 存档存在哪里？**
A: 浏览器 localStorage。清除浏览器数据会丢失存档。

## 📈 进阶技巧

### 伤害计算公式
```
最终伤害 = (基础伤害 + 力量) × 易伤(1.5) × 姿态(怒火×2/神格×3) - 格挡
最终受伤 = (基础伤害 + 敌力量) × 虚弱(0.75/纸鹤0.5) × 易伤(1.5) × 怒火(×2) - 格挡
```

### 卡组效率理论
- **10-15张**：最优，核心牌上手率高
- **16-20张**：可接受，需要过牌支撑
- **20+张**：过载，需要大量过牌或检索

### X费卡牌使用技巧
- 穿刺(X费)在有能量药水或遗物加能量的情况下效果爆炸
- 使用前确保剩余能量最大化

### 遗物联动
- **手里剑 + 双节棍**：每3次攻击同时+1力和+1能
- **死灵书 + 无惧疼痛**：消耗牌得随机牌+格挡
- **木乃伊手 + 腐化**：打出能力减费→更多能力
