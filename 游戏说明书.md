# 尖塔之魂 · 完全游戏说明书

## 上卷 · 新手教程篇

---

### 第一章：什么是"尖塔之魂"？

**尖塔之魂**是一款卡牌构筑 Roguelike 游戏。每次攀登尖塔都是一次全新的冒险——你从零开始构筑牌组，每走一步都面临选择，每一次死亡都是学习，每一次胜利都是对策略的验证。

**一句话概括**：爬上 45 层的尖塔，用卡牌战斗，打败每层的怪物，收集更强的卡牌和遗物，最终击败第 45 层的最终 BOSS。

### 第二章：第一次启动游戏

当你打开游戏，首先看到的是**开始画面**——
- 正上方是彩色渐变的大标题"尖塔之魂"
- 中间是四个角色的选择卡片（带图标和简短描述）
- 下方有三个难度按钮
- 最下面是"开始攀登"按钮

#### 2.1 第一步：选择角色

点击四个角色卡片中的一个来选择角色。你的选择决定了：
- 初始 HP 和卡牌
- 专属卡牌池（游戏中只会出现该角色的专属牌）
- 被动天赋
- 流派选项

> **新手建议**：第一次玩推荐选择**铁甲战士**，他有最高的初始 HP(80) 和简单直接的力量体系，最容易上手。

#### 2.2 第二步：选择流派

选定角色后，会出现**初始牌组流派选择**。每个角色有 4 个流派（包括"基础"），不同流派给你不同的初始遗物和额外起始牌。

例如铁甲战士的流派：
- **力量流**：初始遗物"金刚杵"（每场战斗+2 力量），额外起始牌为燃烧+厚重之刃
- **消耗流**：初始遗物"木乃伊手"，额外起始牌为坚毅+无惧疼痛
- **重装流**：初始遗物"角钉"（+14 格挡），额外起始牌为摔绊+耸肩无视
- **基础**：标准开局，没有额外流派遗物

> **新手建议**：第一次玩选铁甲战士的**力量流**或**基础**流派，力量体系最直观。

#### 2.3 第三步：选择难度

| 难度 | 颜色 | 说明 | 适合人群 |
|------|------|------|---------|
| 🌱 普通 | 绿色 | 标准体验，敌人无额外加成 | 第一次接触的新手 |
| ⚡ 困难 | 橙色 | 敌人更强，但玩家获得首回合增益 | 有经验的玩家 |
| 💀 噩梦 | 红色 | 敌人极强，限制更多 | 寻求挑战的老手 |

> **新手绝对要选普通难度**！困难模式敌人有额外力量，噩梦模式还会限制抽牌数。

#### 2.4 第四步：开始冒险

点击"⛰️ 开始攀登"按钮，正式进入尖塔！

### 第三章：界面导航速成

#### 3.1 顶部状态栏（HUD）

游戏画面最顶部有一条半透明的信息栏，显示：

```
❤️ 当前HP/最大HP  ⚡ 当前能量/最大能量  🪙 金币
当前楼层 📤抽牌堆余量  📥弃牌堆余量  角色名·房间类型·姿态
```

- ❤️ **HP**：你的生命值，归零则游戏结束
- ⚡ **能量**：每回合获得 3 点，打出卡牌消耗能量
- 🪙 **金币**：在商店中消费
- **楼层数**：当前在尖塔第几层
- 📤 **抽牌堆**：剩余可抽的牌数（点击可查看内容）
- 📥 **弃牌堆**：已弃/打出的牌数（点击可查看内容）

#### 3.2 战斗界面布局

```
┌─────────────────────────────────────────────┐
│  ❤️ 80/80  ⚡ 3/3  🪙 45  5F 📤5 📥3    │  ← HUD
├─────────────────────────────────────────────┤
│                                             │
│   [敌人] ← 敌人显示区 (HP条/状态/意图)     │
│                                             │
│   [玩家面板] ← 角色属性、buff/debuff        │
│                                             │
│   [手牌区] ← 你的手牌 (彩色卡牌)            │
│       ◀ [卡牌1] [卡牌2] ... [卡牌N] ▶       │
│         [结束回合] [药水]                     │
└─────────────────────────────────────────────┘
```

#### 3.3 如何操作

- **打出卡牌**：点击手牌即可打出（必须有足够能量）
- **结束回合**：点击"结束回合"按钮
- **使用药水**：战斗中点击药水图标
- **查看牌堆**：点击 📤(抽牌堆) 或 📥(弃牌堆)
- **查看图鉴**：按 **C** 键或点击卡组数量区域
- **查看地图**：按 **M** 键或点击楼层区域
- **查看日志**：按 **L** 键

### 第四章：战斗基础教学

#### 4.1 战斗是怎么进行的？

战斗是回合制的，流程如下：

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│  回合开始   │ →   │  出牌阶段   │ →   │  回合结束   │
│  · 抽5张牌  │     │  · 消耗能量 │     │  · 弃手牌   │
│  · 获得3能量│     │  · 打出卡牌 │     │  · 格挡消失 │
│  · 触发效果 │     │  · 触发效果 │     │  · 触发效果 │
└─────────────┘     └─────────────┘     └─────────────┘
                                              ↓
┌─────────────┐     ┌─────────────┐
│  你的回合   │     │  敌人回合   │
│  (重复)     │ ←   │  · 敌人行动 │
│             │     │  · 触发效果 │
└─────────────┘     └─────────────┘
```

#### 4.2 卡牌详解

每张卡牌包含以下信息：

```
┌─────────────────┐
│  ┌───────────┐  │
│  │  费用: 1   │  │  ← 左上角数字：打出需要消耗的能量
│  │            │  │
│  │  旋风斩    │  │  ← 卡牌名称
│  │            │  │
│  │  对所有敌  │  │
│  │  人造成8点 │  │  ← 效果描述
│  │  伤害      │  │
│  │            │  │
│  │  common    │  │  ← 稀有度
│  │  攻击      │  │  ← 卡牌类型(边框颜色)
│  └───────────┘  │
│     攻击牌       │
│  🔴 暗红边框     │
└─────────────────┘
```

**卡牌边框颜色**：
- 🔴 **暗红色** = 攻击牌 (Attack)：对敌人造成伤害
- 🔵 **暗蓝色** = 技能牌 (Skill)：格挡、抽牌、施加效果
- 🟡 **暗金色** = 能力牌 (Power)：持续整场战斗的增益
- ⚪ **灰色** = 状态牌 (Status)：诅咒类，通常无法打出

#### 4.3 你的第一场战斗

1. **回合开始**：自动抽 5 张牌，获得 3 点能量
2. **看敌人头顶的图标**：
   - ⚔️ 红色剑 = 敌人要攻击你
   - 🛡️ 蓝色盾 = 敌人要防御
   - 💧 紫色水滴 = 敌人要施加负面效果
   - ⬆️ 橙色箭头 = 敌人要给自己加 Buff
   - 数字 = 造成的伤害量或获得的格挡量
3. **选择要打的牌**：
   - 点击费用为 1 的**打击**牌 → 对敌人造成 6 点伤害
   - 点击费用为 1 的**防御**牌 → 获得 5 点格挡（吸收伤害）
4. **结束回合**：点击"结束回合"
5. **看敌人行动**：敌人会根据头顶的意图行动
6. **重复**直到战斗胜利或 HP 归零

#### 4.4 重要的战斗概念

**格挡**：格挡可以完全吸收伤害。但格挡会在你的回合结束时消失（除非你有"壁垒"能力）。

```
敌人攻击 10 点 → 你有 8 格挡 → 实际受到 2 点伤害
敌人攻击 5 点 → 你有 10 格挡 → 完全抵挡，不受伤害（格挡降至 5）
```

**HP 归零** = 游戏结束！必须重新开始。

**战斗胜利**后获得：
- 金币
- 卡牌奖励（从 5 张中选 1 张加入牌组）
- 概率获得药水
- 少量 HP 回复

### 第五章：必知基础规则

#### 5.1 关于能量

- 每回合获得 **3 点能量**（可通过遗物/能力提升上限）
- 每张卡牌左上角的数字是打出所需的能量
- 能量是有限的，所以要合理规划每回合的出牌
- 能量没用完不会累积到下一回合

#### 5.2 关于抽牌

- 每回合开始时抽 **5 张牌**
- 抽牌堆抽空时，弃牌堆会洗入抽牌堆
- 手牌最多 10 张
- 回合结束时剩余的手牌全部弃入弃牌堆

#### 5.3 关于牌组

- 初始牌组有 **10 张牌**（4 打击 + 4 防御 + 2 流派牌）
- 随着游戏进行，你会不断**添加新牌**到牌组中
- 也可以**移除**牌（在商店或事件中）
- 牌组越精简，关键牌上手越稳定

#### 5.4 三个核心原则

1. **能量管理**：每回合只有 3 能量，要最大化每点能量的价值
2. **牌组精简**：20-25 张牌是最优范围，太多会导致关键牌抽不到
3. **攻防平衡**：不能只堆攻击，也不能只堆防御，要根据敌人调整

### 第六章：属性与状态完全手册

#### 6.1 战斗属性表

| 属性 | 默认值 | 作用 | 获取方式 |
|------|--------|------|---------|
| HP | 角色决定 | 归零则游戏结束 | 战斗回复、药水、篝火 |
| 能量 | 3/回合 | 打出卡牌需要消耗 | 某些卡牌、遗物可增加 |
| 格挡 | 0 | 吸收伤害，回合末消失 | 防御牌、技能牌 |
| 力量 | 0 | 每点 +1.5 伤害 | 卡牌、遗物、药水 |
| 敏捷 | 0 | 每点 +1 防御格挡 | 卡牌、遗物、药水 |
| 集中 | 0 | 每点 +0.5 球体效果 | 卡牌、遗物（仅机兵） |
| 念力 | 0 | 10 层 → 进入神格 | 卡牌（仅观者） |

#### 6.2 负面状态（Debuff）

| 状态 | 图标 | 效果 | 持续方式 |
|------|------|------|---------|
| **虚弱** | 💧 | 造成的伤害 ×0.75 | 每回合 -1 层 |
| **易伤** | 💫 | 受到的伤害 ×1.5 | 每回合 -1 层 |
| **中毒** | ☠️ | 每回合造成 层数×2 伤害，然后层数减半 | 层数自然衰减 |
| **脆弱** | 🧊 | (部分敌人对玩家施加的易伤) | 见具体卡牌 |

**虚弱的具体效果**：
- 敌人有虚弱：敌人攻击你时伤害 ×0.75
- 你有虚弱：你对敌人的伤害 ×0.75
- 纸鹤遗物：虚弱效果翻倍（×0.5 而不是 ×0.75）

**易伤的具体效果**：
- 敌人有易伤：你对敌人的伤害 ×1.5
- 手钻遗物：易伤效果翻倍（×2.0 而不是 ×1.5）

**中毒的运作方式**：

```
第 1 回合：施加 6 层中毒 → 回合末敌人受到 6×2=12 伤害，层数变为 3
第 2 回合：再施加 4 层中毒（共 7 层） → 回合末敌人受到 7×2=14 伤害，层数变为 4
第 3 回合：不施加 → 回合末敌人受到 4×2=8 伤害，层数变为 2
```
> **注意**：中毒的伤害在敌人回合开始时触发，每层造成 2 点伤害（非原版的每层 1 点）。

#### 6.3 正面状态（Buff）

| 状态 | 说明 |
|------|------|
| **荆棘** | 你被攻击时，对攻击者造成荆棘值伤害 |
| **人工制品** | 抵挡一次负面效果施加。被消耗后消失 |
| **缓冲** | 抵挡一次伤害（无论多少），消耗一层 |
| **虚无** | 抽到时直接消耗（仅幽魂形态等少数能力） |
| **固有** | 战斗开始时直接上手，不占用首回合抽牌 |

#### 6.4 姿态系统

观者有 3 种姿态和 1 种特殊计量条：

| 姿态 | 进入方式 | 效果 | 退出效果 |
|------|---------|------|---------|
| **怒火** 🔥 | 爆发、暴怒、渐强等 | 伤害 ×2，受伤害 ×1.5 | — |
| **宁静** 💧 | 警惕、冥想等 | — | 退出时 +3(+1) 能量 |
| **神格** ✨ | 满 10 念力自动进入、崇拜等 | 伤害 ×(3+神威)，吸血 50% | 持续 1 回合后退出 |
| **中立** | 默认状态，空手、清醒等退出 | 无效果 | — |

**姿态切换时触发的效果**：
- **精神壁垒**：切换时获得格挡
- **风暴姿态**：切换时全敌 AOE 伤害
- **暗影姿态**：切换时全敌中毒
- **阴阳**：切换时抽牌 + 格挡
- **能量护符**：进入姿态时 +3 格挡 + 抽 1

### 第七章：卡牌系统完全手册

#### 7.1 卡牌稀有度与出现规律

卡牌出现概率受当前幕数影响，幕数越高稀有卡越容易出：

| 稀有度 | 幕1 | 幕2 | 幕3 |
|--------|-----|-----|-----|
| 普通(Common) | 必然出现 | 必然出现 | 必然出现 |
| 稀有(Uncommon) | 75%概率 | 100% | 100% |
| 稀有(Rare) | 25%概率 | 50% | 100% |

> 角色专属卡只会在对应角色的游戏中出现，万花筒碎片遗物可解除此限制。

#### 7.2 卡牌升级系统

在篝火处选择"升级"可以升级一张卡牌。升级后的卡牌：
- 描述中**升级版**（uDesc）的数值生效
- 部分卡牌费用降低（如爆发 2→1 费）
- 部分卡牌获得固有属性

在篝火处，选择你想要升级的卡牌（卡牌必须有"未升级"标记）。

**温馨提示**：优先升级核心输出牌和经常使用的卡牌。

#### 7.3 特殊卡牌机制详解

**X 费牌**：费用标记为 -1 或一个"X"。打出时需要选择消耗多少能量（比如 0~3），然后根据消耗的能量量决定效果。

**消耗**：标记"消耗"的卡牌使用后会从游戏中移除（不进入弃牌堆）。消耗牌的效果通常更强力，但只能用一次。

**固有**：标记"固有"的卡牌战斗开始时直接上手，不占用首回合抽牌。能力牌通常有固有效果。

**虚无**：标记"虚无"的卡牌抽到后如果本回合不使用，会在回合结束时消耗掉。

**多段攻击**：一次攻击造成多次伤害。例如"双斩"造成 6 点伤害 ×2 次。多段攻击时，力量会被分摊：
- 你有 6 点力量，打出双斩（2 段） → 每段伤害 +3（6×1.5÷2）

### 第十一章：伤害机制深度解析

#### 11.1 伤害计算公式（完整版）

```
最终伤害 = 基础伤害计算 → 姿态修正 → 上限裁剪 → 易伤修正 → 格挡吸收

第一步：基础伤害计算
  d = 卡牌基础伤害
  d += (力量 × 1.5) / 命中次数        // 多段攻击力量分摊
  d += 腕刃加成（如果是 0 费攻击牌）
  d += 赤牛/暴怒面具等遗物加成

第二步：虚弱修正
  if (你有虚弱) d = floor(d × 0.75)

第三步：姿态倍率
  倍率 = 1.0
  怒火姿态: 倍率 += 1.0
  神格姿态: 倍率 += (神威层数 - 1)
  幻影杀手: 倍率 += 0.5
  
  限制: 倍率 ≤ 8.0（神格+神威时 ≤ 12.0）

第四步：武器附魔（首张攻击加成）
  d = floor(d × 武器附魔倍率)

第五步：易伤修正（独立乘算）
  if (敌人有易伤) d = floor(d × 1.5)
  手钻遗物: d = floor(d × 2.0)

第六步：荆棘/格挡/反射
  最终伤害 = max(0, d - 荆棘)
  格挡吸收部分伤害
  剩余伤害扣减敌人 HP
```

#### 11.2 伤害计算示例

**示例**：铁甲战士，力量 10，有易伤，打出双斩（6 伤×2），在怒火姿态下

```
对每段攻击：
  基础 = 6
  力量加成 = floor(10 × 1.5 / 2) = 7    // 双斩 2 段，力量分摊
  小计 = 6 + 7 = 13
  虚弱（假设无）→ 13
  姿态倍率: 怒火 → ×(1.0 + 1.0) = ×2.0
  倍率上限检测: 2.0 ≤ 8.0 → 通过
  小计 = floor(13 × 2.0) = 26
  易伤: floor(26 × 1.5) = 39
  
  每段伤害 39，两段共 78 伤害！
```

#### 11.3 格挡吸收机制

```
敌人攻击 20，你有 8 格挡：
  格挡吸收 8 点（格挡消失）
  实际受到 12 点伤害

敌人攻击 20，你有 25 格挡：
  格挡吸收全部 20 点（格挡降到 5）
  实际受到 0 点伤害
```

**减伤顺序**：格挡 → 缓冲 → 相位转移(减半) → 幽魂形态(伤害降为1) → 钨棒(-1) → 鸟居(≤5→1) → 护盾发生器(-5) → 实际扣血

### 第十三章：敌人对策手册

#### 13.1 各 BOSS 对策详解

**史莱姆王（幕1BOSS）**
- 机制：无视格挡！你的格挡完全没用
- 对策：放弃防御，全力输出！带减伤遗物（钨棒、缓冲）比格挡更有用
- 保持 HP 在 50 以上比较安全

**六火亡魂（幕1BOSS）**
- 机制：8 伤×3 的多段攻击很痛
- 对策：每回合保持 15+ 格挡，多段攻击会触发多次荆棘
- 困难/噩梦模式会塞灼烧，带进化卡抵消

**守卫者（幕2BOSS）**
- 机制：每被打就加 8 甲，30% 反射
- 对策：用单次大伤害减少触发次数，如重锤(32 伤)、超光束(26 伤)
- 反射很烦，回血手段要准备好

**铜制机械（幕2BOSS）**
- 机制：越打力量越高，回合末荆棘增生
- 对策：速战速决，越拖越难
- 铜机械也会破盾，别太依赖格挡

**觉醒者（幕3BOSS）**
- 机制：每打能力牌 +1 力量，半血进入二阶段
- 对策：少打能力牌，或在他进入二阶段前秒杀
- 二阶段 +5 力 +5 荆棘，非常致命

**时间吞噬者（幕3BOSS）**
- 机制：先制 12 伤+3 易伤，伤害极高
- 对策：需要大量格挡和减伤遗物
- 最高 34+ 伤害，必须有对应防御手段

#### 13.2 精英敌人对策

**Lagavulin（幕1）**：速战速决，最好 3 回合内解决。

### 第十四章：运营策略指南

#### 14.1 路线选择策略

**优先考虑精英战**：精英虽然危险，但遗物奖励非常宝贵。如果你状态好，尽量走多精英的路线。

**适当安排篝火**：不要等残血才去篝火。如果血量在 70% 以上，优先升级卡牌再休息。

**商店规划**：金币 100+ 时再进商店比较划算，可以买遗物或关键牌。

#### 14.2 牌组管理

**牌组规模**：
- 初期（幕1）：10-15 张
- 中期（幕2）：15-25 张
- 后期（幕3）：20-30 张
- 精减牌组：15-20 张最理想

**什么时候拿牌**：
- 缺输出 → 拿攻击牌
- 缺防御 → 拿格挡牌
- 缺过牌 → 拿抽牌牌
- 出现核心流派牌 → 必拿

**什么时候不拿牌**：
- 牌组已经够强 → 跳过
- 和流派方向不一致 → 跳过
- 重复的基础牌 → 跳过

#### 14.3 关于删牌

优先删除顺序：
1. **伤口/晕眩**（诅咒牌）
2. **打击**（随着游戏进行越来越弱）
3. **防御**（如果防御手段已经充足）

#### 14.4 何时升级什么牌

**优先升级**：
1. 核心输出牌（如旋风斩 8→11、致死伤 20→28）
2. 经常使用的基础牌（如打击 6→9 有易伤）
3. 费用降低的牌（如爆发 2→1 费，彩虹 2→0 费）
4. 获得重要新效果的牌（如双斩升级获得易伤+敏捷）

### 第十五章：无限循环详解

#### 15.1 什么是无限循环？

理论上，你可以通过特定的卡牌组合，在一回合内无限次出牌和过牌，直到击杀所有敌人。这就是"无限循环"。

#### 15.2 铁甲战士无限

**核心**：双持 + 坚毅/耸肩无视
1. 打出能力牌降低费用/获得能量
2. 用坚毅消耗抽到的牌 + 无惧疼痛获得格挡
3. 黑暗之拥补充手牌
4. 循环直到敌人都被磨死

**更可行的无限**：
- 飞踢(若易伤回能+抽1) + 易伤手段 → 0 费打+过牌
- 冲突(0 费) + 适当牌组条件

#### 15.3 观者无限循环

**最经典的无限**：
1. 渐强(进怒火，消耗) + 警惕(进宁静，+3 能量)
2. 怒火状态打攻击牌 → 退出怒火进宁静(+3 能) → 再进怒火 → 再打攻击
3. 能量越打越多

**前提**：需要足够多的姿态切换牌，且牌组精简

#### 15.4 静默猎手无限

**弃牌无限**：
1. 准备(0 费，抽 2 弃 1) + 战术(弃过牌回 1 能)
2. 杂技(抽 3 弃 1) → 策略家(回 1 能)
3. 循环依赖弃牌堆的回收

### 第十七章：常见问题 FAQ

**Q：为什么我的牌有时候灰色不能打？**
A：状态牌（伤口、晕眩）无法打出，只能占据手牌位置。灼烧抽到时自动触发伤害然后消耗。

**Q：格挡为什么有的时候会消失有的时候不会？**
A：壁垒能力让你格挡不清零。正常情况下回合结束格挡会消失。

**Q：什么是 X 费牌？怎么用？**
A：X 费牌打出时会弹出一个选择框选消耗多少能量。如果你选 X=0，效果不会触发。

**Q：为什么打多段攻击感觉伤害变低了？**
A：多段攻击时力量会被分摊，每段获得的力量加成 = floor(力量 × 1.5 / 段数)。所以 10 力量打 5 段的话每段只 +3 伤害。这是为了避免多段攻击过于 OP。

**Q：小刀算不算攻击牌？**
A：小刀是攻击牌（type='attack'），会触发手里剑、饰品扇等"每 X 张攻击牌"的效果。

**Q：如何获得遗物？**
A：精英战斗、宝箱、商店购买、部分随机事件。

**Q：可以在战斗中存档吗？**
A：游戏会自动存档（localStorage），关闭页面重开会显示"继续冒险"按钮。

**Q：如何查看当前牌组里有什么牌？**
A：按 C 键打开图鉴，或点击结算画面的卡组数量文字。
