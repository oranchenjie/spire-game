### 伤害系统 (185-293)
| 函数 | 行号 | 功能 |
|------|------|------|
| `dmgEnemy(e,b,g)` | 187 | 对敌人造成伤害（核心函数） |

| `dmgPlayerD(g,r)` | 290 | 自残伤害（触发铁血被动） |

**伤害公式** (dmgEnemy):
```
d = base + strength×1.5/hitCnt + 遗物加成
d = d × (weak ? 0.75 : 1.0)
dmgMult = 1.0 + (wrath ? 1.0) + (divinity ? divineWrath-1)
cap = ×8 (divinity+wrath ×12)
d = × dmgMult  (capped)
d = × vulnerable (1.5 or 2.0 with handDrill)
```

### 敌人系统 (1127-1199)
`const ET=[]` (1130) — 敌人模板

**模板格式**: `{id, name, hb, hs, act, mv, elite, boss, preemptive, react, _reflect, _phaseTwo, _lifeLink, _nullifyBlock, eotStr, eotHeal}`

- `hb`: 基础HP, `hs`: HP每层成长系数
- `mv`: 动作序列 `[{t:'a/d/db/b/m/s/h', d, b, n, ef, v}]`
- `t='a'=攻击`, `'d'=防御`, `'db'=Debuff`, `'b'=Buff`, `'m'=多段`, `'s'=塞牌`, `'h'=治疗`
- `t='e'`=抽能, `t='f'`=反馈(按出牌数伤)
- **新能力**: `_enrageHp`(狂怒阈值), `_lifesteal`(吸血比例), `_curseOnHit`(攻击塞伤口), `_retaliate`(反击伤害), `_lastStand`(死亡AOE)
- **怪物创意增强 2025-06-03**: 24处敌人改动，详见下方

**`playCard` 流程**:
1. 计算费用（腐化、医疗箱、剔骨特殊处理）
2. 扣能量、移出手牌
3. 触发遗物（腕刃、赤牛、手里剑、苦无、双节棍等）
4. 调用 `d.f(g, enemies, card)` 执行卡牌效果
5. 触发千刀万剐、残影等
6. 处理消耗、丢入弃牌堆
7. 杀敌触发（恶魔形态击杀+力）

**`startTurn` 流程**:
1. 能量恢复(+冰淇淋保留)、格挡重置
2. Buff/Debuff衰减（虚弱、易伤）
3. 萝卜/生姜免疫
4. 回合开始触发（金属化、铁壁之力、恶魔形态、毒雾等）
5. 无限刀锋、创造性AI、你好世界
6. 计算抽牌数 `dc`（基础5，困难-1，蛇戒+1等）
7. 抽牌

## 平衡参数速查

**敌人生成** (`mkEnemy`, 1581):
```
hp = (hb + floor * hs * hpBase) * diffHp
dmg = (base + floor * dScale) * diffDmg
hpBase = 1.0 + act * 0.45
dScale = act1=0.60, act2=0.80, act3=1.00
bScale = act1=0.30, act2=0.40, act3=0.50
diffHp = 1 + diff * 0.30
diffDmg = 1 + diff * 0.35
```

**卡牌稀有度权重**:
```
rareW = act1=0.30, act2=0.55, act3=1.0
uncomW = act1=0.80, act2=1.0, act3=1.0
```

## 注意事项
1. 文件使用 tab 缩进，Edit 工具可能因缩进不匹配而失败，优先用 Python heredoc 做批量替换
2. 所有中文文本在 Python 中需用 UTF-8 编码
